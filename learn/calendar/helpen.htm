Instructions to upload your schedule
If you are ready to upload your schedule which you have finished editing to this schedule, please follow the format as follows. When you save the file, save it with csv format so that you can upload it to this schedule.
Examples 2004-4-1%% 08:30%% 09:00%% week%%2004-5-1%%login%% 2%% TEST1%% CALENDAR TEST1 2004-4-2%% 10:00%% 11:00%% %% %% %% 0%% TEST2%% CALENDAR\nTEST2
Notes YYYY-MM-DD%%Start Time%%Due Time%%Period Start Date%%Period Due Date%%Notification%%Notification ahead of Due Date%%Title%%Contents
Remarks
  1. [YYYY-MM-DD]: (Start) Date of the event, with the format of YYYY-MM-DD (Year, month and date are all connected with a "-".)
  2. [Start Time], [Due Time]: Time length of the event, with the format of HH:MM (Both H and M have to be integels.)
  3. The value of [Period] can be either [space], date, week or month.
  4. [Period Due Date]: with the format the same as that in the date of the event. If Period Start Date is shown empty, this column is invalid.
  5. The value of [Notification] can be either [space], login, email or login|email. (Login indicates the notification shown when you log in, email indicates the notification via email, and login|email indicates that both are required.)
  6. The value of [Notification ahead of Due Date] has to be not less than 0 (integels), with 0 indicating the notification of the day. (If the notification is shown empty, this column is invalid.)
  7. In [Contents], use "\n" for line feed.
