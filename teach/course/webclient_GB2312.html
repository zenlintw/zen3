如何设定使用 Web 资料夹
Windows 7 与 Windows 8 使用者
  1. 安装 WebClient 注册表
    • 下载 WebClient.reg 注册表

    • 直接点选【运行】

    • 允许程序对此计算机进行更改，点【是】

    • 是否确定要继续，点【是】

    • 已经成功加到注册表中，点【确定】



  2. 新增「受信任的站点」
    • 开启IE，单击『工具』>『Ｉnternet 选项』>『安全』

    • 单击『受信任的站点』，再单击『站点』

    • 取消【勾选】『对该区域中的所有站点要求服务器验证(https:)』
      输入平台站点，http://
      点【添加】


  3. 重新开机电脑


  4. 开始使用 Web 资料夹
    • 到课程办公室＞课程管理＞教材上传，于 Web 资料夹，点【开启】

    • 当开启资源管理器后，则可以将所需档案复制、粘贴或拖拉方式上传
注意事项
  • 目前 Web资料夹，只限定使用 IE 浏览器
    支援浏览器　：IE8 (含)以上
    支援作业系统：Windows 7、Windows 8

  • Web 资料夹上传档案，单一最大档案限制 4GB

  • 不可以使用 InPrivate 方式浏览，此会造成 Web 资料夹无法开启

  • 若发生在 Web 资料夹，点【开启】无反应，请尝试关闭所有 IE 浏览器后，再重新开启 IE 或重开机。

  • 若粘贴档案时，出现『错误 0x80070780: 系统无法访问此文件。』
    表示 Web资料夹 已与平台失去连线，请关闭 IE 与 Web资料夹后，再重新执行一次
