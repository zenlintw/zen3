If you want to use FTP to upload, please follow the steps as follows.

1.Open the FTP software on your computer, for example : AbsoluteFTP, BlazeFtp,CuteFTP, SmartFTP ...etc。
2.Fill in the FTP address with the elearning web site address.
3.Fill in the FTP username with your current logon ID.
4.Fill in the FTP password with your current logon password.
5.If need be, please fill in the FTP port number with default 21 port to connect.
6.Start the connection.
※Notes :
1.You must login the elearning web site first and go into this function. Then you can use FTP software to upload files to the course. If you just open FTP software and do not login the elearning web site, you can't use FTP to upload course contents.
2.The file directory you login the ftp is your course content directory. If you want to upload contents to other courses, please choose that course on the web site and reconnect the FTP.
